# Bemindlabs Loopduck

## How do I install these formulae?

`brew install bemindlabs/loopduck/<formula>`

Or `brew tap bemindlabs/loopduck` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "bemindlabs/loopduck"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
